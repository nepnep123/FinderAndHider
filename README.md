# FinderAndHider
## 1. 개요
* 숨바꼭질게임을 모티브로 하여 VR 환경에서 숨는사람, 찾는사람으로 나뉘어 숨는사람은 해당 맵에 사물로 변하여 제한시간동안 찾는사람에게 걸리지 않고 
 숨는다. 찾는사람은 제한시간동안 사물로 변한 숨는사람을 찾으면 승리를 하게 된다.
